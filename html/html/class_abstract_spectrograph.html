Descrição Detalhada

The AbstractSpectrograph class provides a generic interface for a spectrograph drawing widget.

Periodically, mainwindows calculates somewhere a FFT transform of audio input signal and extracts its spectrum. This spectrum may be offered to objects of subclasses of AbstractSpectrograph for drawing nice things

Moreover, left and audio mean levels are also communicated, so level bars (or something like that) can be drawn.
